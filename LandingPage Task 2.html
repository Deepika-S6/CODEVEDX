What you'll practice
The building blocks of the web

Every great website starts with the same fundamentals. This project covers all of them.

🧱

Semantic structure

Use header, main, section, and footer tags to give your page structure that browsers and screen readers understand.

📐

Layout with CSS

Arrange elements using Flexbox and Grid — the modern tools for building columns, rows, and responsive sections.

🎨

Color & typography

Choose a palette, pick font sizes, and control spacing so your page feels intentional, not accidental.

📦

The box model

Padding, margin, and border — understanding how every element occupies space is the key to clean designs.

🔗

Buttons & links

Style interactive elements with hover states, focus rings, and cursor changes for a polished result.

📱

Responsive basics

A couple of media queries go a long way. Make sure your page looks great on a phone, not just a laptop.

Project breakdown
What to build, section by section

Break the page into clear parts and tackle one at a time.

01

Navigation bar

A sticky header with a logo on the left and links on the right using Flexbox space-between.

02

Hero section

A bold headline, short description, and call-to-action button with generous padding.

03

Feature cards

A three-column grid of cards — each with an icon, title, and description. CSS Grid makes this easy.

04

About / story block

A two-column layout: image on one side, text on the other. Practice vertical alignment.

05

Call-to-action banner

A full-width colored section with a headline and button. Contrast makes it stand out.

06

Footer

Links, copyright text, and a logo. Keep it consistent with the rest of your palette.

The process
From blank file to live page in 4 steps

No setup drama. Open your editor and follow along — you'll have something real on screen within the hour.

📄1

Create your files

Make an index.html and a style.css in the same folder. That's your entire project.

🏗️2

Build the structure

Write the HTML skeleton — nav, hero, sections, footer. Don't worry about looks yet; focus on structure.

🎨3

Style with CSS

Link your stylesheet and start adding colors, fonts, spacing, and layout. See changes instantly in the browser.

📱4

Make it responsive

Add a couple of media queries. Resize your browser window to check that everything looks great on every screen.
